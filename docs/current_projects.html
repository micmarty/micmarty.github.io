TensorHive open-source

Lightweight computing resource management tool for executing machine learning applications in distributed TensorFlow

We've just started making an open-source solution for distributed trainings in TensorFlow. It will:

  • Launch with a single command
  • Have a dashboard built with Material Design components (Google Android style)
  • Communicate over ssh
  • Monitor resources like: GPU, memory load, etc.
The project has started within GradientPG (Student Research Group I participate in).
